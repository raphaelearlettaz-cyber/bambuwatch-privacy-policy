# bambuwatch-privacy-policy
La politique de confidentialité pour l'application BambuWatch
Politique de confidentialité — BambuWatch
Dernière mise à jour : 23 mars 2026

1. Collecte des données
BambuWatch ne collecte, ne stocke et ne partage aucune donnée personnelle sur ses serveurs.

L'app se connecte à votre compte Bambu Lab via l'API officielle de Bambu Lab. Vos identifiants de connexion sont uniquement transmis à Bambu Lab et ne transitent jamais par nos serveurs.

2. Données stockées sur votre appareil
Les informations suivantes sont stockées localement sur votre iPhone, uniquement pour le bon fonctionnement de l'app :

Votre token d'authentification Bambu Lab
Les préférences de l'app
Ces données ne quittent jamais votre appareil et ne sont pas partagées avec des tiers.

3. Services tiers
BambuWatch utilise l'API de Bambu Lab pour récupérer les données de vos imprimantes. Les données échangées sont régies par la politique de confidentialité de Bambu Lab.

4. Notifications
Si vous activez les notifications, celles-ci sont générées localement sur votre appareil ou via les services de notification d'Apple (APNs). Aucune donnée personnelle n'est transmise.
